# 🛡️ Fodubu Security Monitor

A lightweight **DevOps monitoring system** for tracking website uptime, latency, and server health in real time.

---

## 🚀 Features

- 🌐 Website uptime monitoring
- ⚡ Response time (latency) tracking
- ❌ Automatic error detection (timeouts, downtime)
- 📝 Logging system
- ⏱️ Scheduled health checks
- 🔌 Ready for server integration

## 🧠 What this tool does

Fodubu Security Monitor continuously checks your server and reports:

- If the site is **UP or DOWN**
- Response time in milliseconds
- Error messages (timeouts, failures)
- Logs everything for analysis
